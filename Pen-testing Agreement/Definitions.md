# Definitions

**Agreement**  
This Penetration Testing Services Agreement between CyberNetsec and Martisato Losgistics

**Client (Martisato Losgistics)**  
The organization receiving penetration testing services.

**Provider (CyberNetsec)**  
The entity performing penetration testing services.

**Confidential Information**  
All non-public information disclosed during the engagement.

**SOW (Statement of Work)**  
A document describing specific engagement scope, deliverables, timelines, and fees.

**In-Scope Assets**  
Systems explicitly identified for testing within an SOW.

**Out-of-Scope Assets**  
Systems that must not be tested unless added explicitly in writing.

**PHI / PII**  
Protected health information or personally identifiable information.

**Cloud Provider Policies**  
Rules enforced by AWS or other vendors governing security testing.

**Good-Faith Security Testing**  
Testing performed with authorization for the purpose of improving security.

**Kill Switch**  
A process to immediately pause testing if unexpected instability occurs.

**Breakglass Access**  
Temporary emergency access provided to restore test execution if blocked by IAM controls.
